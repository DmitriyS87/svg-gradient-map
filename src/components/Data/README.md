# Data

Contains default data:

* `primitivesData` — list of initial primitives
* `presetsData` — list of initial presets
* `primitivesAttrs` — object with primitives attributes
* `palettes` — list of palettes for Duotone filter

Usage:

```js static
import {
  primitivesData,
  primitivesAttrs,
  presetsData,
  palettes} from '../Data';
```
